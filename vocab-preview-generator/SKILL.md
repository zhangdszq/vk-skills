# 单词预习资源生成

为少儿英语教培公司批量生产单词预习资源。输入课程脚本，输出结构化词表、发音音频、词义配图，全部按课程结构归档。

## 输入

- 课程脚本 Excel 文件，包含以下字段：
  - `Level`（课程级别）
  - `Unit`（单元编号）
  - `sn`（课程节点编号，如 `OW-L0-U1-LC1-1`）
  - 课程文本描述（包含本节教学词汇）

## 输出

```text
target/
├── L{n}_全部词汇.xlsx
├── 01 音频/
│   └── Level {n}/
│       └── {sn}/
│           └── {word}.mp3
└── 02 图片/
    └── Level {n}/
        └── {sn}/
            └── {word}.jpeg
```

## 五步生产流程

### 第一步：词汇抽取

从课程文本里提取当前节点**明确作为教学内容出现**的词汇。

抽取规则：
- 只提取课程文本中明确标注为教学词汇的词
- 排除：字母、句型、发音练习标签、说明语、练习题标题
- 不猜测、不扩展

示例：
- `OW-L0-U1-LC1-1` → 提取 `am, it`
- `OW-L0-U1-LC1-2` → 提取 `good, great, wonderful, hello, goodbye`

### 第二步：词条结构化

给每个词补齐以下字段，输出标准词条：

| 字段 | 说明 |
|------|------|
| `单元` | Unit 编号 |
| `课程节点` | sn 编号 |
| `新课标编号` | 同 sn |
| `词汇类别` | 动词 / 名词 / 形容词 / 功能词等 |
| `词汇内容` | 单词本身 |
| `词汇解码` | 音节拆分或自然拼读标注 |
| `词性` | 词性 |
| `词义` | 中文词义 |
| `例句` | 含目标词的例句，目标词用 `<>` 标注 |

示例词条：

```text
新课标编号: OW-L0-U1-LC1-2
词汇内容: great
词义: 很棒的
词性: 形容词
例句: It is <great>.
```

将所有词条汇总输出为 `L{n}_全部词汇.xlsx`。

### 第三步：生成发音音频

使用有道词典发音接口批量下载 `.mp3`：

```text
https://dict.youdao.com/dictvoice?type=1&audio={word}
```

归档路径：`target/01 音频/Level {n}/{sn}/{word}.mp3`

优化策略：
- 同一个词在多个节点重复出现时，复用已有音频，不重复下载

### 第四步：生成词义配图

使用图片生成接口为每个词生成 `.jpeg` 图片。

**核心原则：图片围绕词义生成，不按例句字面生成。**

各类词汇的图片语义设计参考：

| 词汇类型 | 图片语义方向 |
|----------|------------|
| 情绪 / 态度词（如 `great`）| 人物正向情绪场景，如开心的孩子、竖起大拇指 |
| 基础功能词（如 `am`）| 简洁、低龄、画面单一 |
| 交通 / 物品类（如 `bus`）| 明确物体主体，无背景干扰 |
| 动物类（如 `cat`）| 高辨识度动物主体，风格活泼 |
| 数字 / 抽象词（如 `ten`）| 单独设计具体可视化场景 |

归档路径：`target/02 图片/Level {n}/{sn}/{word}.jpeg`

### 第五步：稳定性处理

批量运行时需处理以下三类问题：

**重复词复用**
- 先检查目标路径是否已有同名文件
- 有则直接复制复用，跳过接口调用
- 保证同一词在不同节点风格一致

**接口限流**
- 图片接口触发限流时自动退避等待
- 支持重试，失败词汇记录到待补跑列表
- 全批次跑完后可单独补跑失败词汇

**特殊词提示词优化**
- 数字词、功能词、抽象词使用独立提示词模板
- 提示词需更明确、更具体，减少内容审核失败概率

## 使用规则

1. 抽取词汇时严格遵守"只提取明确教学词汇"的边界，有疑问时优先不提取
2. 词条结构化时所有字段必须填写，不留空
3. 图片生成时以词义为核心，例句仅作辅助参考
4. 每次运行前先检查已有资源，避免重复生成
5. 批量任务跑完后输出资源总结，包含：生成数量、跳过数量、失败数量、失败词汇列表

## 注意

- 词汇抽取精准度直接决定后续所有资源的质量，这一步不能草率
- 图片生成是最容易出现质量偏差的环节，抽象词和功能词务必单独处理
- 结构化词表本身是可独立交付的教研资产，应确保字段完整、格式规范
