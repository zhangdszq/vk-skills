# 设置与个性化

## 操作步骤：打开设置

1. 在侧栏最底部找到用户头像图标
   - 已登录：显示用户头像或名字首字母
   - 游客模式：显示「G」头像和「游客模式」标签
2. 点击头像图标
3. 弹出「设置」对话框
4. 左侧有 7 个标签页，右侧显示对应设置内容
5. 修改完成后点击右下角「完成」保存

## 个性化（第 1 个标签）

### 操作步骤：设置用户名称和工作描述

1. 打开设置 → 点击左侧「个性化」标签（人像图标）
2. 在「姓名」输入框中填入你的名字
   - 设置后，首页问候语会显示你的名字（如"Yangyi 晚上好"）
3. 在「工作描述」输入框中描述你的工作内容
   - AI 会根据此信息更好地理解你的需求和习惯
4. 在「全局提示词」输入框中添加对 AI 的通用指令（可选）
   - 这里的提示词会自动附加到所有对话中
5. 点击「完成」保存

## 通用设置（第 2 个标签）

### 操作步骤：设置全局唤起快捷键

1. 打开设置 → 点击左侧「通用」标签（齿轮图标）
2. 找到「快捷键唤起小窗」设置项
3. 点击当前快捷键显示区域（默认显示 ⌘W）
4. 按下新的快捷键组合（如 ⌘+Space）
5. 新快捷键立即生效，在任何应用中都可按此快捷键唤起牛马AI快速输入窗口

### 操作步骤：设置语音输入快捷键

1. 打开设置 → 点击左侧「通用」标签
2. 找到「语音输入快捷键」设置项
3. 先打开旁边的开关（启用语音快捷键功能）
4. 点击快捷键显示区域
5. 按下想要的快捷键组合
6. 设置完成后，在任何界面中按住该快捷键即可开始录音，松开自动转文字

### 操作步骤：配置其他通用选项

1. 打开设置 → 点击左侧「通用」标签
2. 根据需要调整以下开关：
   - **菜单栏**：是否在系统菜单栏显示应用图标
   - **开机自动启动**：是否在电脑登录时自动启动牛马AI
   - **通知**：是否接收系统桌面通知
   - **自动检查更新**：是否自动检测新版本
   - **自动下载更新**：检测到新版本后是否自动下载
3. 还可以配置：
   - **权限模式**：普通模式（需确认）/ YOLO 模式（自动执行）
   - **命令白名单**：允许 AI 自动执行的命令列表

### 操作步骤：恢复默认设置

1. 打开设置 → 点击左侧「通用」标签
2. 点击左下角的「恢复默认」按钮
3. 所有通用设置恢复为初始值

## 数据管理（第 5 个标签）

### 操作步骤：导出对话备份

1. 打开设置 → 点击左侧「数据」标签（导入图标）
2. 找到「导出对话」按钮
3. 点击后选择保存位置
4. 等待导出完成，生成备份文件

### 操作步骤：导入对话备份

1. 打开设置 → 点击左侧「数据」标签
2. 找到「导入对话」按钮
3. 选择之前导出的备份文件
4. 等待导入完成，对话自动恢复到侧栏列表中

### 操作步骤：设置自动备份

1. 打开设置 → 点击左侧「数据」标签
2. 找到「数据库备份」区域
3. 设置「备份目录」：点击选择备份文件存放位置
4. 设置「备份频率」：
   - 手动：仅在你手动触发时备份
   - 每天：每日自动备份一次
   - 每周：每周自动备份一次
5. 系统自动保留最近 3 个备份，旧备份自动清理

### 数据安全说明
- 所有对话数据存储在本地 SQLite 数据库
- 不上传用户对话内容到服务器
- 支持完全离线使用（配合本地模型）

## 语音模型设置（第 6 个标签）

### 操作步骤：下载和配置语音识别模型

1. 打开设置 → 点击左侧「语音模型」标签（喇叭图标）
2. 页面显示可用的语音识别模型
3. 如果模型未下载，点击「下载」按钮
4. 等待模型下载完成（模型在本地运行，保障隐私）
5. 下载完成后自动启用
6. 可查看语音识别历史记录

## 账户（第 7 个标签）

### 操作步骤：查看账户信息

1. 打开设置 → 点击左侧「账号」标签（退出图标）
2. 显示当前用户信息：头像、昵称、邮箱、手机号、用户名

### 操作步骤：退出登录

1. 打开设置 → 点击左侧「账号」标签
2. 点击「退出登录」按钮
3. 退出后进入游客模式

### 游客模式说明
- 无需登录即可使用大部分功能
- 以下功能受限：
  - 不可查看反馈历史
  - 不可发布技能到牛马市场
