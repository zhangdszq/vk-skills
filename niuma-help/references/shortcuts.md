# 快捷操作

## 快捷键一览

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 唤起快速输入窗口 | ⌘W（可自定义） | 全局快捷键，任意应用中可用 |
| 语音输入 | 可自定义 | 按住说话，松开自动转文字 |
| 页内查找 | ⌘F / Ctrl+F | 在当前对话中搜索文字 |
| 发送消息 | Enter | 发送当前输入的消息 |
| 换行 | Shift+Enter | 在输入框中换行 |
| 关闭弹窗 | Esc | 关闭当前打开的对话框/弹窗 |

## 快速输入窗口

轻量级的浮动输入窗口，适合快速提问，无需切换到主窗口。

### 操作步骤：自定义唤起快捷键

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「通用」标签
3. 找到「快捷键唤起小窗」设置项
4. 点击当前快捷键显示区域（默认 ⌘W）
5. 按下新的快捷键组合
6. 新快捷键立即生效

### 操作步骤：使用快速输入窗口

1. 在任意应用中按下全局快捷键（默认 ⌘W）
2. 屏幕中央弹出浮动的输入小窗
3. 小窗界面说明：
   - 顶部：「牛马AI」标识和关闭按钮 (X)
   - 中间：文本输入框
   - 右侧：MCP 服务器选择图标 + 发送按钮
   - 底部提示：↵ 发送 · ⇧↵ 换行 · ESC 关闭
4. 输入问题后按 Enter 发送
5. AI 回复在输入框下方展示，并显示「查看完整对话」链接
6. 点击「查看完整对话」可跳转到主窗口继续对话

### 操作步骤：隐藏快速输入窗口

- 再次按 ⌘W（或你设定的快捷键）
- 或按 Esc
- 或点击窗口右上角 X 关闭

## 托盘图标

### 操作步骤：使用系统托盘图标

1. 牛马AI 运行后，在 macOS 菜单栏 / Windows 系统托盘区域会显示应用图标
2. **单击图标**：切换快速输入窗口的显示/隐藏
3. **右键图标**：打开托盘菜单
   - **显示主窗口**：打开/聚焦到应用主窗口
   - **退出**：关闭应用

### 操作步骤：开启/关闭菜单栏显示

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「通用」标签
3. 找到「菜单栏」开关
4. 打开：应用图标显示在系统菜单栏/托盘
5. 关闭：不显示托盘图标

## 语音输入

### 操作步骤：首次设置语音输入

1. **下载语音模型**（首次使用需要）：
   - 点击侧栏底部的头像图标 → 打开「设置」
   - 点击左侧「语音模型」标签
   - 点击「下载」按钮下载语音识别模型
   - 等待下载完成（模型在本地运行，保障隐私）
2. **设置语音快捷键**：
   - 在设置中点击左侧「通用」标签
   - 找到「语音输入快捷键」设置项
   - 打开开关（启用语音快捷键）
   - 点击快捷键显示区域，按下想要的快捷键组合
3. 点击「完成」保存设置

### 操作步骤：使用全局语音快捷键输入

1. 确保已完成上述首次设置
2. 在任意界面中（即使牛马AI在后台也可用）：
   - **按住**你设定的语音快捷键
   - 屏幕上弹出语音指示窗口，表示正在录音
   - 开始说话
   - **松开**快捷键
3. 系统自动将语音转为文字
4. 转写结果自动填入当前对话的输入框中
5. 检查文字无误后按 Enter 发送

### 操作步骤：使用输入框中的麦克风按钮

1. 在对话输入框右侧找到麦克风按钮（🎤 图标，位于模型选择器左侧）
2. 点击麦克风按钮开始录音
3. 按钮上方出现录音时长计时器
4. 再次点击麦克风按钮停止录音
5. 按钮显示转写中的加载动画
6. 转写完成后文字自动填入输入框

> 如果系统提示麦克风权限被拒绝，需要前往系统设置 → 隐私与安全 → 麦克风，为牛马AI开启权限。

## 自动更新

### 操作步骤：配置自动更新

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「通用」标签
3. 找到以下两个开关：
   - **自动检查更新**：开启后，应用启动时自动检查是否有新版本
   - **自动下载更新**：开启后，检测到新版本自动在后台下载
4. 当更新下载完成后：
   - 应用顶部出现更新提示横幅
   - 点击提示重启应用
   - 重启后自动安装更新

### 操作步骤：开机自动启动

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「通用」标签
3. 找到「开机自动启动」开关
4. 打开后，每次电脑开机登录时，牛马AI 自动启动并最小化到托盘
