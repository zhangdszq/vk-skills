# 长期计划（项目管理）

## 概述

长期计划是牛马AI的核心特色功能。用户通过对话描述目标，AI 自动规划、拆解任务，并在能力范围内自动执行——真正实现 AI 当牛马。

## 创建项目

### 操作步骤：创建一个新项目

1. 在侧栏中找到「长期放养」区域（位于功能按钮下方、最近对话上方）
2. 点击该区域右侧的 **+** 按钮
3. 弹出「创建项目」对话框，包含以下内容：
   - **项目名称**：输入项目名称（必填）
   - **选择模板**：从预设模板中选择（可选），有 5 个模板：
     - 社媒内容创作（社交媒体图标）
     - 个性化学习（学习图标）
     - 投资案例跟踪（投资图标）
     - 信息分析洞察（分析图标）
     - 编程项目（编程图标）
   - **初始描述**：自由输入你想做的事情，如"帮我规划一个 AI 科普公众号"
   - **附加文件**：可选择添加相关参考文件
   - **选择技能**：可选择为项目启用特定技能
4. 点击「创建」按钮
5. 自动跳转到项目视图，并进入聊天 Tab
6. 新创建的项目出现在侧栏「长期放养」列表中

### 操作步骤：进入已有项目

1. 在侧栏「长期放养」区域找到目标项目
2. 点击项目名称
3. 主区域切换为该项目的视图

## 项目视图

创建或进入项目后，主区域显示项目视图，顶部有 5 个 Tab 标签可切换。

### 1. 聊天 Tab

项目专属的 AI 对话区域。与普通对话的区别：

- AI 自动获得项目上下文（计划文档、任务列表、依赖关系）
- AI 可以调用项目专用工具（创建任务、更新计划等）
- 支持多个对话（如"初始规划""第一周复盘""调整内容策略"）
- 左侧有项目对话列表，可切换或新建

**操作步骤：与项目 AI 对话**
1. 进入项目视图，默认在「聊天」Tab
2. 左侧显示项目对话列表，右侧是对话区
3. 点击左侧列表中的对话可切换；点击 **+** 新建项目对话
4. 在输入框中描述你的目标或需求
5. AI 会根据项目上下文回复，并可能自动：
   - 调用 `update_plan_document` 生成项目计划文档
   - 调用 `create_tasks` 批量创建任务（含日期、执行类型、依赖关系）
6. 确认 AI 的规划后，自动任务会按日程开始执行

### 2. 排期 Tab

**操作步骤：查看和管理排期**
1. 点击项目视图顶部的「排期」Tab
2. 页面呈左右分栏布局：
   - **左侧 - 计划文档**：AI 生成的 Markdown 项目计划，可直接编辑
   - **右侧 - 周视图日历**：以日历形式展示任务安排
3. 日历中的任务卡片按日期排列，高度根据预估耗时调整
4. 当 AI 更新计划时，左侧文档上方会弹出通知条，提示有更新

**操作步骤：在日历中手动添加事项**
1. 在排期 Tab 的右侧日历区域
2. 点击目标日期的空白区域
3. 填写事项名称和详情
4. 保存后事项出现在日历中

### 3. 任务 Tab（看板）

**操作步骤：查看和管理任务**
1. 点击项目视图顶部的「任务」Tab
2. 进入 Kanban 看板视图，分为四列：
   | 待办 (Todo) | 进行中 (In Progress) | 已完成 (Done) | 审核 (Review) |
   |-------------|---------------------|---------------|---------------|
3. 每张任务卡片显示：标题、计划日期（📅）、执行类型（🤖自动/👤手动）、优先级颜色条

**操作步骤：拖拽管理任务状态**
1. 按住任务卡片
2. 拖拽到目标列（如从"待办"拖到"进行中"）
3. 松开鼠标，任务状态自动更新

**操作步骤：编辑或操作单个任务**
1. 右键点击任务卡片，弹出菜单：
   - **编辑**：修改任务标题、描述、日期等
   - **删除**：移除该任务
   - **设置优先级**：紧急（红色）/ 高（主题色）/ 普通（灰色）
   - **立即执行**：手动触发自动任务立即执行
2. 或者点击任务卡片，弹出详情弹窗，可查看：
   - 任务状态和依赖关系
   - 执行记录（何时开始、何时完成）
   - 产出文件列表

### 4. 文件 Tab

**操作步骤：浏览项目文件**
1. 点击项目视图顶部的「文件」Tab
2. 左侧显示项目工作区的文件目录树
3. 点击文件夹可展开/收起
4. 点击文件名，右侧打开文件预览面板
5. 在预览面板中可点击「在系统应用中打开」用默认程序打开文件

### 5. 设置 Tab

**操作步骤：配置项目通知**
1. 点击项目视图顶部的「设置」Tab
2. 找到「任务完成通知」区域：
   - 打开通知开关
   - 选择通知频道（飞书/企微/自定义 Webhook）
   - 选择详情级别：简要 / 详细
3. 找到「手动任务提醒」区域：
   - 开启后，手动任务到期时会发送提醒

**操作步骤：添加通知频道（如飞书 Webhook）**
1. 在项目设置 Tab 中找到「通知频道管理」
2. 点击「添加频道」
3. 选择类型：飞书 Webhook / 企业微信 Webhook / 自定义 Webhook
4. 填写 Webhook URL
5. 点击「测试」验证连通性
6. 保存频道配置

## 自动执行引擎

这是长期计划的核心能力。

### 工作原理

1. **定时轮询**：每 60 秒检查是否有到期的自动任务
2. **条件检查**：计划日期已到 + 所有前置依赖任务已完成
3. **自动执行**：创建独立对话，AI 根据任务描述和项目上下文自动完成工作
4. **依赖解锁**：任务完成后，自动检查并触发下游依赖任务

### 任务类型

- **自动任务**（🤖）：AI 全自动执行，用户无需干预
- **手动任务**（👤）：到期时发送提醒通知，由用户自行完成

### 执行控制

- 单个任务最大执行时长：10 分钟
- 串行队列：同一时刻只执行一个任务
- 失败重试：可配置最大重试次数

**操作步骤：手动触发任务执行**
1. 在任务 Tab（看板）中找到目标任务
2. 右键点击任务卡片
3. 选择「立即执行」
4. 任务状态变为「进行中」，AI 开始执行
5. 执行完成后任务自动移到「已完成」列

**操作步骤：取消正在执行的任务**
1. 在任务 Tab 中找到正在执行的任务（在「进行中」列）
2. 右键点击任务卡片
3. 选择「取消执行」

### 通知方式

任务完成或失败时，通过以下方式通知：
- 桌面系统通知
- 飞书/企微/自定义 Webhook（需在项目设置中配置）
- 应用内通知（如果用户正在查看该项目）

### 任务依赖

支持任务间的依赖关系：
- 任务 B 依赖任务 A → 任务 A 完成后，任务 B 才会开始
- 支持链式依赖（A → B → C）
- AI 创建任务时会自动设置合理的依赖关系

## AI 可用的项目工具

在项目对话中，AI 拥有 7 个专用工具：

| 工具 | 功能 |
|------|------|
| create_tasks | 批量创建任务（含依赖关系、日期、执行类型） |
| update_task | 更新任务属性（标题、描述、状态、优先级等） |
| mark_task_done | 标记任务完成，附带产出文件列表 |
| update_plan_document | 更新项目计划文档（Markdown） |
| list_tasks | 查询任务列表（可按状态和日期筛选） |
| add_dependency | 添加任务依赖关系 |
| read_project_file | 读取工作区文件（查看前置任务产出） |
