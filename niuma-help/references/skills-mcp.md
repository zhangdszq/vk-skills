# 技能系统与 MCP

## 技能系统

技能（Skill）是牛马AI的扩展能力系统，类似插件。每个技能赋予 AI 特定领域的专业能力。

### 什么是技能

技能是一组预定义的指令和工具配置，安装后 AI 可以使用该技能完成特定类型的任务。例如：
- 公众号文章写作技能
- 数据分析技能
- 图片生成技能
- PPT 制作技能
- 视频创作技能

### 操作步骤：浏览和安装技能（牛马市场）

1. 点击侧栏顶部第四个按钮「雇佣牛马」（商店图标）
2. 主区域切换为牛马市场页面
3. 顶部有两个标签：
   - **市场**：浏览所有可用的社区技能
   - **已安装**：管理已安装的技能
4. 使用搜索栏输入关键词搜索技能，或使用分类筛选下拉菜单
5. 找到想要的技能后，点击技能卡片查看详情（描述、版本、安装次数）
6. 点击「安装」按钮，一键安装到本地
7. 安装完成后可在「已安装」标签中查看

### 操作步骤：从本地安装技能

1. 进入牛马市场页面（侧栏「雇佣牛马」按钮）
2. 点击页面右上角的 **···**（更多菜单）
3. 选择安装方式：
   - **从 ZIP 文件安装**：选择本地 ZIP 压缩包
   - **从文件夹安装**：选择本地技能文件夹
4. 等待安装完成

### 操作步骤：在对话中启用技能

1. 在对话输入框左侧工具栏中找到「技能」按钮（右上角显示已启用数量的徽章）
2. 点击弹出技能选择面板
3. 面板列出所有已安装的技能，每个技能有开关
4. 勾选你需要在本次对话中使用的技能
5. 关闭面板，已选技能以标签形式显示在输入框上方的附件栏中
6. 正常对话即可，AI 会根据需要自动调用已启用的技能

### 操作步骤：通过斜杠命令快速使用技能

1. 在输入框中输入 `/`（斜杠）
2. 弹出技能快速选择菜单，列出可用技能命令
3. 选择目标技能
4. 技能指令自动填入输入框
5. 补充你的具体需求后按 Enter 发送

### 操作步骤：卸载技能

1. 进入牛马市场页面（侧栏「雇佣牛马」按钮）
2. 切换到「已安装」标签
3. 找到要卸载的技能
4. 点击技能卡片上的「卸载」按钮

### 操作步骤：发布自己的技能

1. 进入牛马市场页面
2. 点击页面右上角的 **···** 更多菜单
3. 选择「上传技能」
4. 选择技能文件/文件夹
5. 填写技能描述信息
6. 提交后等待管理员审核，审核通过后在牛马市场中公开
7. 可在牛马市场中查看自己发布的技能状态和安装统计

## MCP（模型上下文协议）

MCP 是一种标准化的方式来扩展 AI 的工具能力。

### 什么是 MCP 服务器

MCP 服务器为 AI 提供额外的工具和数据源。例如：
- 搜索引擎（Brave Search）
- 网页读取（Jina Reader）
- 代码仓库访问（GitHub）
- 学术论文（ArXiv）
- 视频信息（YouTube）
- 自定义 API 集成

### 内置 MCP 服务器

应用预置了常用的 MCP 服务器，可以一键启用：

| 服务器 | 功能 |
|--------|------|
| Brave Web Search | 网络搜索 |
| Jina Reader | 网页内容读取 |
| GitHub | GitHub 仓库操作 |
| ArXiv | 学术论文搜索 |
| YouTube | 视频信息获取 |
| Grok | Grok AI 集成 |

### 操作步骤：启用内置 MCP 服务器

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「MCP」标签
3. 页面显示所有内置 MCP 服务器列表
4. 找到想要启用的服务器（如 Brave Web Search）
5. 点击该服务器旁的开关，切换为启用状态
6. 部分服务器需要填写 API Key（如 Brave Search 需要 API 密钥）
7. 点击「完成」保存设置

### 操作步骤：添加自定义 MCP 服务器

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「MCP」标签
3. 点击「添加自定义服务器」按钮
4. 填写配置信息：
   - **服务器名称**：给服务器起个名字
   - **配置参数**：JSON 格式的服务器配置（包含命令、参数等）
5. 如果服务器需要 npm 依赖，点击「安装依赖」按钮
6. 点击「保存」
7. 新服务器出现在列表中，默认启用

### 操作步骤：在对话中选择 MCP 服务器

1. 在对话输入框左侧工具栏中找到「MCP」按钮（链接图标，右上角显示已启用数量）
2. 点击弹出 MCP 服务器选择面板
3. 列出所有已配置且启用的 MCP 服务器
4. 勾选你需要在本次对话中使用的服务器
5. 关闭面板
6. 正常对话即可，AI 会根据需要自动调用已启用的 MCP 工具
7. 工具调用过程和结果会在消息中展示

### 操作步骤：删除 MCP 服务器

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「MCP」标签
3. 找到要删除的服务器
4. 点击该服务器旁的删除按钮
5. 确认删除
