# 反馈与帮助

## 反馈入口

应用右下角有一个绿色的圆形浮动反馈按钮（💬），始终显示在最顶层——即使打开了设置等弹窗也能看到并点击。鼠标悬停时显示「反馈」文字标签。

## 操作步骤：提交反馈

1. 点击应用右下角的绿色圆形按钮（💬）
2. 弹出「反馈」对话框
3. 选择反馈类型：
   - **Bug 报告**：报告你遇到的问题或错误
   - **功能建议**：提出新功能想法
4. 在描述框中详细描述问题或建议（最多 5000 字）
5. 添加截图（可选，最多 5 张）：
   - 点击 📷 按钮：截取当前屏幕内容（包括弹窗内容，因为反馈按钮浮于最顶层）
   - 点击 📁 按钮：从本地选择图片上传
6. 点击「提交」按钮
7. 反馈发送成功

> 无需登录即可提交反馈（匿名提交）。系统会自动附带你的操作系统和应用版本信息，方便开发团队排查问题。

## 操作步骤：查看反馈历史

> 需要登录才能查看反馈历史。

1. 点击应用右下角的绿色圆形按钮（💬）
2. 在反馈对话框中，切换到「我的反馈」标签页
3. 显示所有已提交的反馈列表
4. 每条反馈显示：
   - 反馈标题和描述
   - 当前状态：待处理 → 处理中 → 已解决 / 已关闭 / 暂缓处理
5. 点击某条反馈展开详情
6. 底部可查看与管理员的对话记录（消息线程）
7. 管理员回复会有「管理员」标签标识
8. 你可以继续回复补充信息

## 首页提示

应用首页（空对话状态）会显示滚动提示条，包含使用技巧：

- ⌘W 可以随时唤醒 AI 输入
- 在对话中满意产出，可以告知 AI 转化长期任务
- 点击右下角绿色按钮，随时反馈问题与建议
- 支持离线模式，在设置中可添加本地模型
- 可以设置语音快捷键，随时语音输入
- 利用工作区路径，可以设置到剪映素材库
