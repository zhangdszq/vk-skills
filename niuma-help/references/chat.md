# AI 对话

## 基本对话

- 在输入框中输入消息，按 Enter 发送
- Shift+Enter 换行
- 支持多轮连续对话，AI 会记住上下文
- 每个工作区下可创建多个独立对话
- 对话自动保存，按时间分组显示（今天/昨天/更早）

### 操作步骤：新建对话

1. 点击侧栏顶部第一个按钮「新建对话」（加号图标）
2. 主区域切换到空白对话界面
3. 在底部输入框中输入你的问题
4. 按 Enter 发送，AI 开始回复

### 操作步骤：切换对话

1. 在侧栏「最近对话」区域找到目标对话
2. 点击对话标题即可切换过去
3. 如果对话较多，可使用侧栏顶部第二个按钮「搜索」全文搜索历史对话

## 消息格式与渲染

AI 回复支持以下富文本格式：

- **Markdown**：标题、粗体、斜体、列表、引用、表格等
- **代码块**：语法高亮，支持多种编程语言
- **LaTeX 数学公式**：行内和块级公式
- **Mermaid 图表**：流程图、时序图、甘特图等
- **SVG/Canvas 可视化**：图形绘制
- **HTML 预览**：交互式 HTML 内容

## 文件附件

### 支持的文件类型
- 文本/代码文件：txt, md, py, js, ts, json, yaml, html, css 等
- 图片：PNG, JPG, GIF, WebP（带预览）
- 文档：PDF, DOCX, XLSX, PPTX, CSV
- 数据库：SQLite（可浏览表结构和数据）
- 视频：支持播放预览

### 操作步骤：添加文件附件

**方式一：点击附件按钮**
1. 点击输入框左侧工具栏中的 📎（回形针）按钮
2. 在弹出的文件选择器中选择文件
3. 文件出现在输入框上方的附件栏中（显示文件名，可点 X 移除）
4. 输入消息后按 Enter 连同附件一起发送

**方式二：拖拽文件**
1. 从文件管理器（Finder / 资源管理器）中选中文件
2. 拖拽到对话区域
3. 松开鼠标，文件自动添加到附件栏

**方式三：粘贴图片**
1. 复制一张图片（截图或从其他应用中复制）
2. 在输入框中按 Ctrl/Cmd+V
3. 图片预览出现在附件栏中

**方式四：@引用工作区文件**
1. 在输入框中输入 `@`
2. 弹出工作区文件列表
3. 选择要引用的文件，文件会作为附件添加

## 文件预览

对话中提到的或 AI 生成的文件可以直接预览：

| 文件类型 | 预览能力 |
|----------|----------|
| CSV/XLSX | 表格形式展示 |
| PPTX | 幻灯片导航浏览 |
| PDF | 文档阅读器 |
| 图片 | 缩略图 + 尺寸信息 |
| 视频 | 播放器 |
| HTML | 沙箱渲染 |
| 代码文件 | 语法高亮 |
| SQLite | 表浏览器 + 查询 |

### 操作步骤：预览 AI 产出的文件

1. AI 回复中如果生成了文件，消息中会显示文件卡片
2. 点击文件卡片，右侧弹出预览面板
3. 根据文件类型自动选择对应的预览方式
4. 可以点击「在系统应用中打开」用默认程序打开文件

### 操作步骤：查看 AI 修改的文件

1. AI 修改文件后，右侧面板会自动弹出并切换到「Changes」标签
2. 面板中列出所有被修改的文件
3. 点击文件名可查看修改内容（diff 对比）

## 扩展思考（Extended Thinking）

AI 可以展示推理过程，帮助用户了解回答的思路。

### 操作步骤：调整思考预算

1. 在输入框左侧工具栏中找到 💡（灯泡）按钮
2. 点击后弹出思考预算选择菜单
3. 选择一个级别：
   - **关闭**：不展示思考过程
   - **低**：简短思考
   - **中**：适度思考
   - **高**：深度推理，适合复杂问题
4. 选择后立即生效，后续 AI 回复会按选定级别展示思考过程
5. 思考过程在回复中以可折叠区块形式显示，点击可展开/收起

## 工具调用

AI 在对话中可以使用多种工具完成任务：

### 文件操作
- **读取文件**：查看文件内容
- **写入文件**：创建新文件
- **编辑文件**：修改已有文件（显示 diff 对比）
- **查找文件**：按模式搜索
- **搜索内容**：在文件中搜索文本

### 命令执行
- **终端命令**：执行 Shell 命令，显示输入和输出
- 命令状态显示：运行中 / 完成 / 错误

### 网络操作
- **网络搜索**：搜索互联网信息
- **获取网页**：读取网页内容

### 权限控制

**操作步骤：配置权限模式**
1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「通用」标签
3. 找到「权限模式」设置项
4. 选择模式：
   - **普通模式**：AI 执行危险操作时需要你手动确认
   - **YOLO 模式**：AI 自动执行所有操作（需谨慎开启）
5. 可在「命令白名单」中配置允许自动执行的命令

## 对话管理

### 操作步骤：搜索对话

1. 点击侧栏顶部第二个按钮「搜索」（放大镜图标）
2. 弹出搜索对话框
3. 输入关键词，自动搜索所有对话内容
4. 点击搜索结果即可跳转到对应对话

### 操作步骤：删除对话

1. 在侧栏「最近对话」中找到目标对话
2. 右键点击对话标题
3. 在弹出的右键菜单中选择「删除」

### 操作步骤：页内查找

1. 在对话页面按 Cmd+F（Mac）或 Ctrl+F（Windows）
2. 顶部弹出查找栏
3. 输入搜索内容，匹配项高亮显示
4. 使用上/下箭头切换匹配项，显示当前/总匹配数

## 消息队列

支持连续发送多条消息，系统会按顺序处理。

### 操作步骤：连续发送消息

1. 在 AI 回复过程中，你可以继续输入新消息
2. 按 Enter 发送，消息进入队列（输入框上方显示队列状态）
3. 也可以点击输入框右侧的「加入队列」按钮（列表加号图标）
4. AI 会按顺序依次处理队列中的消息
5. 最多支持 5 条消息排队

### 操作步骤：取消 AI 回复

1. 在 AI 正在回复时，输入框右侧的发送按钮变为「停止」按钮（方块图标）
2. 点击即可中断当前回复

## 在对话中使用技能

### 操作步骤：启用/选择技能

**方式一：通过工具栏**
1. 点击输入框左侧工具栏中的「技能」按钮（齿轮图标，右上角显示已启用数量）
2. 弹出技能选择面板，列出所有已安装的技能
3. 勾选要在本次对话中使用的技能
4. 关闭面板，选中的技能显示在附件栏中

**方式二：通过斜杠命令**
1. 在输入框中输入 `/`
2. 弹出技能快速选择菜单
3. 选择要调用的技能
4. 技能指令自动填入输入框

## 在对话中使用 MCP

### 操作步骤：启用 MCP 服务器

1. 点击输入框左侧工具栏中的「MCP」按钮（链接图标，右上角显示已启用数量）
2. 弹出 MCP 服务器选择面板
3. 勾选要在本次对话中使用的 MCP 服务器
4. AI 会根据对话需要自动调用已启用的 MCP 工具

## 切换模型

### 操作步骤：在对话中切换 AI 模型

1. 在输入框右侧找到模型选择器（显示当前模型名称和提供商图标）
2. 点击打开下拉菜单
3. 模型按提供商分组显示
4. 选择目标模型，立即切换
5. 后续对话将使用新选择的模型
