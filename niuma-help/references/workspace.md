# 工作区管理

## 多工作区

牛马AI 支持多个工作区，每个工作区关联一个本地文件夹路径。

### 工作区隔离

每个工作区拥有独立的：
- 对话历史
- 长期计划/项目
- 定时任务
- 技能配置
- MCP 服务器配置

### 工作区路径

工作区路径是 AI 文件操作的根目录。AI 可以在此目录下读写文件、执行命令。

**使用场景提示**：可以将工作区路径设置为剪映素材库、代码项目目录、写作文件夹等。

### 操作步骤：创建/添加新工作区

1. 在主区域顶部找到工作区标签栏（显示当前已打开的工作区标签）
2. 点击标签栏右侧的 **+** 按钮
3. 弹出「添加工作区」对话框，有两个选项：
   - **打开文件夹**：选择一个已有的本地文件夹作为工作区
   - **创建新项目**：新建一个文件夹作为工作区
4. 选择或创建文件夹后，新工作区标签出现在标签栏中
5. 自动切换到新工作区

### 操作步骤：切换工作区

1. 在主区域顶部的工作区标签栏中
2. 直接点击目标工作区的标签名称即可切换
3. 切换后，侧栏的对话列表、项目列表等都会更新为该工作区的内容

### 操作步骤：关闭/管理工作区标签

1. 将鼠标悬停在工作区标签上
2. 点击标签右侧出现的 **X** 关闭按钮
3. 关闭标签只是从标签栏移除，不会删除工作区数据
4. 拖拽标签可调整工作区顺序

### 操作步骤：修改工作区路径

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「通用」标签
3. 找到「工作区路径」设置项
4. 点击路径旁的「修改」按钮
5. 在弹出的文件夹选择器中选择新路径
6. 确认后工作区指向新目录

## 终端

内置终端功能，无需切换到外部终端。

### 操作步骤：打开终端面板

1. 在对话页面底部的输入框左侧工具栏中
2. 找到终端按钮（终端图标，第 5 个按钮）
3. 如果有终端在后台运行，按钮右上角会显示运行中的终端数量
4. 点击按钮，底部弹出终端面板

### 操作步骤：管理终端会话

1. 终端面板打开后，顶部显示终端标签页
2. 点击标签页中的 **+** 创建新终端会话（最多 5 个）
3. 点击不同标签切换终端会话
4. 拖拽标签可调整顺序
5. 点击标签上的 **X** 关闭该终端
6. 拖拽终端面板顶部边缘可调整面板高度

### 操作步骤：关闭终端面板

1. 再次点击输入框工具栏中的终端按钮
2. 终端面板收起隐藏（终端会话继续在后台运行）

## 定时任务

独立于长期计划的轻量级定时任务系统。

### 操作步骤：进入定时任务管理

1. 点击侧栏顶部第三个按钮「定时任务」（时钟图标）
2. 主区域切换为定时任务管理界面
3. 显示当前工作区的所有定时任务列表

### 操作步骤：创建定时任务

1. 在定时任务管理界面中，点击「新建任务」按钮
2. 填写以下内容：
   - **任务名称**：给任务起个名字
   - **执行内容**：描述 AI 要执行的操作
   - **Cron 表达式**：设置执行频率（如 `0 9 * * *` 表示每天 9 点执行）
3. 确认关联的工作区（默认为当前工作区）
4. 点击「创建」保存

### 操作步骤：管理已有定时任务

1. 在任务列表中找到目标任务
2. 可进行的操作：
   - **开关切换**：点击任务旁的开关，启用或禁用该任务
   - **手动触发**：点击「立即执行」按钮，不等定时触发直接运行一次
   - **查看历史**：点击任务展开，查看历次执行记录和状态
   - **删除任务**：点击删除按钮移除任务

## 搜索

### 操作步骤：全文搜索对话

1. 点击侧栏顶部第二个按钮「搜索」（放大镜图标）
2. 弹出搜索对话框
3. 在搜索框中输入关键词
4. 搜索结果按时间分组显示（今天/昨天/更早）
5. 点击搜索结果即可跳转到对应对话

### 操作步骤：当前对话内查找

1. 在任意对话页面中按 Cmd+F（Mac）或 Ctrl+F（Windows）
2. 页面顶部弹出查找栏
3. 输入要查找的文字
4. 页面中匹配内容高亮显示
5. 查找栏显示「当前第 X 个 / 共 Y 个匹配」
6. 点击上/下箭头或按 Enter 在匹配项之间跳转
7. 按 Esc 关闭查找栏
