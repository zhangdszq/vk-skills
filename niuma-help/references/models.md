# 模型与 AI 提供商配置

## 模型选择

在对话中可以随时切换 AI 模型。

### 操作步骤：在对话中切换模型

1. 在对话输入框右侧找到模型选择器（显示当前模型名称和提供商 Logo）
2. 点击下拉菜单
3. 模型按提供商分组显示（如 Claude 系列在 Anthropic 分组下）
4. 点击目标模型即可切换
5. 后续对话将使用新选择的模型

### 默认可用模型

- **Anthropic Claude**（默认推荐）：最强综合能力
- **Sonnet**：更快速的响应
- **Haiku**：最快速、适合轻量级任务
- 其他已配置提供商的模型

## AI 提供商

牛马AI 支持多个 AI 服务提供商，用户可以根据需要添加和切换。

### 内置提供商

- **Anthropic**（默认）：Claude 系列模型

### 可添加的提供商

| 提供商 | 说明 |
|--------|------|
| MiniMax | MiniMax AI 模型 |
| Kimi | 月之暗面 Kimi 模型 |
| 智谱 (Zhipu) | 智谱 AI GLM 模型 |
| 火山方舟 (Volcengine) | 字节跳动 AI 模型 |
| OpenRouter | 多模型路由平台 |
| Ollama | 本地运行的开源模型 |
| SiliconFlow | 硅基流动 AI 模型 |
| ZenMux | 多模型聚合平台 |
| 自定义提供商 | 任意 OpenAI 兼容的 API |

### 操作步骤：添加 AI 提供商

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「模型」标签
3. 页面显示已有的提供商列表
4. 点击「添加提供商」按钮
5. 从列表中选择提供商类型（如 Kimi、OpenRouter、Ollama 等）
6. 填写必要信息：
   - **API Key**：在提供商官网获取的密钥（Ollama 不需要）
   - **Base URL**：自定义提供商需要填写 API 地址
7. 点击保存
8. 新提供商的模型会出现在对话中的模型选择器下拉菜单中

### 操作步骤：设置默认模型

1. 点击侧栏底部的头像图标 → 打开「设置」
2. 点击左侧「模型」标签
3. 在提供商列表中找到目标提供商
4. 点击该提供商下的模型列表
5. 选择想要设为默认的模型
6. 点击「设为默认」

### 操作步骤：配置离线模式（使用 Ollama）

> 通过添加 Ollama 提供商，可以完全离线使用 AI 对话。

1. **先在电脑上安装 Ollama**：
   - 前往 [ollama.com](https://ollama.com) 下载安装
   - 打开终端运行 `ollama pull llama3`（或其他模型）下载模型
   - 确认 Ollama 服务已启动（默认运行在 localhost:11434）
2. **在牛马AI中添加 Ollama 提供商**：
   - 点击侧栏底部的头像图标 → 打开「设置」
   - 点击左侧「模型」标签
   - 点击「添加提供商」
   - 选择「Ollama」
   - Base URL 默认为 `http://localhost:11434`，无需修改
   - 不需要填写 API Key
   - 保存
3. **切换到本地模型**：
   - 回到对话界面
   - 点击输入框右侧的模型选择器
   - 在下拉菜单中找到 Ollama 分组
   - 选择已下载的本地模型
4. 现在可以断网使用 AI 对话

> 提示：离线模式下不可用联网功能（网络搜索、网页获取等）。
